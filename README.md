
# AI Interview Trainer 🎤

An AI-powered interview preparation tool designed to help students and job seekers practice technical interviews.

The application asks AI/ML interview questions, collects responses, evaluates answers, and provides feedback to improve interview performance.

## 🚀 Features

- AI/ML interview questions
- User answer collection
- Basic answer evaluation
- Interview scoring system
- Personalized feedback

## 🛠 Technologies Used

- Python
- Google Colab

## 🎯 Goal

To build an intelligent interview preparation assistant that helps users improve their confidence and technical interview skills.
